GitHub-Hexo-Next搭建免费独立个人博客

本文是我的第一篇正式博客，想来还是挺兴奋的！之前看了下其他平台的博客，我觉得都太乱了！所以我决定自己通过GitHub+Hexo+Next搭建独立个人博客，免费独立易操作，

主要是看起来非常简洁！让人看着非常舒服。所以我花了一天半的时间，做了这样一件事

教程:

教程一如何搭建一个独立博客——简明Github Pages与Hexo教程
教程二GitHub+Hexo+Next搭建免费独立个人博客
教程三如何利用GitHub Pages和Hexo快速搭建个人博客

问题：

下面简单说下我遇到的问题，而教程上没有的：

1、-$ npm install -g hexo-cli安装不成功，是因为没有到对应的根目录下启动git

2、-Localhost:4000没有作用是因为端口号被占用，通过$ hexo server -p 5000,修改端口号即可，在本地搭建

3、-部署不成功：3.0之后版本的是git+ssh地址，还要注意空格

eg: deploy:

type: git

repository: git@github.com:BigTuTu/bigtutu.github.io.git

4、分类、标签等功能不成功：在hexo文件下的source文件里分别创建categories、tags文件，分别在各自文件夹下创建index.md文件，定义category，tags等。在每一篇发表的博文头部自己定义这篇博文的分类、标签等等，系统会自动生成。

总结：

我选择的是next主题：修改完善，可以参考官网：http://theme-next.iissnan.com/

个人的博客的基本雏形已经有了，还有很多细节可以优化，我在后面学习中慢慢完善。
